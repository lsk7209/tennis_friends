## 컨텐츠 특징

모든 선수 페이지는 다음 요소를 포함합니다:

✅ **SEO/AEO/GEO 최적화**
- 메타데이터 (title, description, keywords)
- Open Graph 태그
- Twitter 카드
- Canonical URL
- 구조화된 데이터 (JSON-LD)

✅ **가독성 최적화**
- 질문형 H2 제목
- 직답 문단 (110자 이내)
- 아이콘과 배지
- 인포 박스와 하이라이트
- 시각적 구분선

✅ **필수 섹션 (7개)**
1. 도입 - 왜 주목받는 선수인가?
2. 정체성 - 어떤 유형의 플레이어인가?
3. 강점 구조 - 세계 상위권을 지키는 힘
4. 대표 경기·명장면
5. 팬들이 좋아하는 이유
6. 최근 경기력 흐름
7. 한 문장 요약

✅ **선택 섹션 (템플릿 타입별)**
- 성장 스토리 (narrative)
- 라이벌 구도 (rivalry)
- 부상과 복귀 (comeback)

✅ **추가 기능**
- 목차 (TOC)
- FAQ 섹션 (3-5개)
- 관련 링크
- CTA 섹션
