# Tech Stack (Web Mode)
- **Framework:** Next.js (App Router)
- **Styling:** TailwindCSS + Shadcn/UI
- **Database:** Supabase (Project specific)
- **Hosting:** Vercel

# Architecture Plan
- **State Management:** React Context / Zustand (if needed)
- **Testing:** Playwright for E2E
- **Component Design:** Flat hierarchy, composition over inheritance.
- **Agentic Workflow:**
  1. Verify current site status (Browser Agent).
  2. Identify missing features or bugs.
  3. Implement changes using `web-ui` skills.
